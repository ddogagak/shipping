K-Packet PDF → CSV/XLSX 테스트 v9

eBay packing slip PDF를 업로드하면 주문번호 기준으로 분리해서 기존 K-Packet CSV 70개 컬럼 포맷으로 출력합니다. 1차 버전이라 저장 DB는 없고, 화면에서 확인/수정 후 다운로드하는 방식입니다.

기본 입력 방식: eBay Orders Report CSV를 읽어 Order Number 기준으로 주문을 그룹화합니다.
PDF import는 fallback입니다. 페이지 단위로 읽어서 여러 페이지에 걸친 주문도 이어 붙입니다.
붙여넣기 방식은 PDF보다 레이아웃 정보가 적어서 금액 파싱이 덜 안정적일 수 있습니다.
기본 규칙: Pipedream 규칙 이식 / 비영어권 특수문자 ASCII 변환 / 기존+추가 K-Packet 국가 반영 / TAX·IOSS 존재 시 subtotal 가격 사용, 없으면 비-US는 15 고정 / 개수는 Quantity 합산.
